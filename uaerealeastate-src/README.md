# bunn-realestate-landing

Marketing site for BUNN — stablecoin payment links for UAE real estate developers and brokers.

## Stack
- Vite + React 19 + TypeScript
- three.js + @react-three/fiber + @react-three/drei (3D dot globe hero)

## Develop
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy
Connected to Vercel project `bunn-realestate-landing`. Pushes to `main` deploy automatically.
